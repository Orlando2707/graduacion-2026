# Invitación de Graduación 2026

Sitio web estático listo para publicarse gratis con GitHub Pages.

## Publicación en GitHub Pages

1. Inicia sesión en GitHub.
2. Crea un repositorio nuevo, por ejemplo: `graduacion-2026`.
3. Déjalo como **Public**.
4. No agregues README, licencia ni `.gitignore` al crearlo.
5. Descomprime este ZIP.
6. Sube **el contenido de la carpeta del proyecto**, no la carpeta contenedora:
   - `index.html`
   - `styles.css`
   - `script.js`
   - carpeta `assets`
   - `README.md`
7. En el repositorio abre:
   **Settings → Pages**
8. En **Build and deployment** selecciona:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
9. Guarda. GitHub publicará la página en una dirección similar a:
   `https://TU-USUARIO.github.io/graduacion-2026/`

## Importante

Para que la vista previa en WhatsApp muestre correctamente la imagen de portada, después de publicar conviene reemplazar en `index.html`:

`<meta property="og:image" content="assets/hero.webp">`

por la URL completa publicada, por ejemplo:

`<meta property="og:image" content="https://TU-USUARIO.github.io/graduacion-2026/assets/hero.webp">`
